## 一、测试范围

本测试用例覆盖 ERP 系统所有前端页面的所有元素，包括：

1. 登录页面 `/login`
2. 商品管理页面 `/inventory/products`
3. 库存概览页面 `/inventory/overview`
4. 规格管理页面 `/inventory/specs`
5. 销售开单页面 `/sales/create`
6. 订单列表页面 `/sales/orders`
7. 订单详情页面 `/sales/orders/:id`
8. 客户列表页面 `/customers`
9. 客户详情页面 `/customers/:id`
10. 应收账款页面 `/finance/receivables`
11. 销售报表页面 `/reports/sales`
